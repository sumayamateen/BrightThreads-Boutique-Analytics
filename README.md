## Framework Applied — OSEMN  

### Obtain & Scrub  
- Defined **SMART business objectives** and **KPIs**  
- Identified relevant **data sources** and collection methods  
- Performed **data validation** and cleaning operations  
- Resolved issues: duplicates, missing values, inconsistent formats, and incorrect entries  

### Explore & Model  
- Conducted **exploratory data analysis (EDA)** on sales transactions  
- Created **data visualizations** to identify trends and patterns  
- Applied **statistical models** — linear regression and clustering  
- Engineered **features** to enhance analytical depth  
- Built **predictive models** for sales forecasting  

### Interpret & Present  
- Translated analytical results into **actionable insights**  
- Developed a **stakeholder presentation deck**  
- Proposed **growth strategies** aligned with KPIs  
- Created an **implementation roadmap** with measurable outcomes
